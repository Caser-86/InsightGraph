# InsightGraph 配置说明

## 产品路径

- `offline`：默认 deterministic 测试 / CI 路径
- `live-llm`：轻量 live 路径
- `live-research`：当前正式 live 产品路径

## 搜索与抓取

关键变量：

- `INSIGHT_GRAPH_USE_WEB_SEARCH`
- `INSIGHT_GRAPH_SEARCH_PROVIDER`
- `INSIGHT_GRAPH_SEARCH_LIMIT`
- `INSIGHT_GRAPH_SEARCH_PROXY`
- `INSIGHT_GRAPH_SERPAPI_KEY`
- `INSIGHT_GRAPH_FETCH_RENDERED`

支持 provider：

- `mock`
- `duckduckgo`
- `google`
- `serpapi`

## GitHub / SEC / 新闻

- `INSIGHT_GRAPH_USE_GITHUB_SEARCH`
- `INSIGHT_GRAPH_GITHUB_PROVIDER=live`
- `INSIGHT_GRAPH_USE_NEWS_SEARCH`
- `INSIGHT_GRAPH_USE_SEC_FILINGS`
- `INSIGHT_GRAPH_USE_SEC_FINANCIALS`

## 研究预算

- `INSIGHT_GRAPH_MAX_TOOL_CALLS`
- `INSIGHT_GRAPH_MAX_FETCHES`
- `INSIGHT_GRAPH_MAX_EVIDENCE_PER_RUN`
- `INSIGHT_GRAPH_MAX_TOKENS`
- `INSIGHT_GRAPH_REPORT_INTENSITY`
- `INSIGHT_GRAPH_MAX_TOOL_ROUNDS`

## 异步任务与持久化

- `INSIGHT_GRAPH_RESEARCH_JOBS_BACKEND`
- `INSIGHT_GRAPH_RESEARCH_JOBS_SQLITE_PATH`
- `INSIGHT_GRAPH_RESEARCH_JOBS_STARTUP_WORKER`
- `INSIGHT_GRAPH_RESEARCH_JOBS_TERMINAL_RETENTION_DAYS`
- `INSIGHT_GRAPH_CHECKPOINT_BACKEND`
- `INSIGHT_GRAPH_CHECKPOINT_RESUME`
- `INSIGHT_GRAPH_POSTGRES_DSN`

## 长期记忆

- `INSIGHT_GRAPH_MEMORY_BACKEND`
- `INSIGHT_GRAPH_USE_MEMORY_CONTEXT`
- `INSIGHT_GRAPH_MEMORY_WRITEBACK`
- `INSIGHT_GRAPH_EMBEDDING_PROVIDER`
- `INSIGHT_GRAPH_EMBEDDING_MODEL`

## 观测

- `INSIGHT_GRAPH_LLM_TRACE`
- `INSIGHT_GRAPH_LLM_TRACE_PATH`
- `INSIGHT_GRAPH_LLM_TRACE_FULL`

## Live Benchmark 说明

Do not commit generated live benchmark reports.
