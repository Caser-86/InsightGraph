# 行为准则

## 我们的承诺

为了创建一个开放和欢迎的社区，我们承诺：

- 尊重所有参与者，无论其背景、身份或经历如何
- 创造一个安全、包容的环境
- 欢迎不同观点和想法
- 专注于对项目和社区最有益的内容

## 预期行为

参与者应该：

- 使用包容的语言
- 接受建设性批评
- 专注于项目最佳利益
- 对其他社区成员表示同理心
- 尊重不同的意见和经验水平

## 不可接受的行为

不可接受的行为包括：

- 骚扰、霸凌或歧视性言论/行为
- 针对个人的人身攻击
- 性骚扰或不当的性内容
- 故意破坏或破坏他人工作
- 发布他人隐私信息（未经同意）
- 其他被合理认为不适合专业环境的行为

## 执法

违反本行为准则的行为可能导致：

1. **警告**: 首次较轻微违规，维护者会发出私下警告
2. **限制**: 多次违规或严重违规，临时或永久限制项目参与权限
3. **删除**: 对于严重或持续的违规，内容可能被删除

## 报告问题

如果您遭遇或目睹了不可接受的行为：

1. **私下联系** 项目维护者（通过邮件或 GitHub）
2. **提供详情**:
   - 事件描述
   - 相关人员
   - 时间和地点（在线或离线）
   - 证据（如截图或链接）

3. **保密**: 我们会保持报告人的隐私

维护者承诺：
- 立即进行调查
- 尽快回应报告者
- 保护举报人免受报复

## 范围

本行为准则适用于：

- 项目仓库（Issues、Pull Requests、讨论）
- 官方通信渠道（邮件列表、论坛等）
- 代表项目的官方活动（会议、研讨会等）

## 灵感来源

本行为准则改编自：

- [Contributor Covenant](https://www.contributor-covenant.org/)
- [Python 社区行为准则](https://www.python.org/psf/code-of-conduct/)

## 变更

本行为准则可能随时更新。主要变更会通过 README 或公告方式告知社区。

感谢帮助我们建立一个友好、尊重的社区！
